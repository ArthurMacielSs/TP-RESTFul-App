TAGS

um tipo de agrupamento, utilizado para pesquisar tipos de noticias, cada noticia tem suas tags que agrupam os conteudos, para pesquisar todas as tagas disponiveis use o comando tags

q

Solicite tags contendo exatamente este texto livre

web-title

Solicite tags começando com este texto livre

exemplo

search?q=tecnologia&tag=science/innovation&web-title=Innovation

type

Retornar apenas tags desse tipo

section

Retornar apenas tags nessas seções

reference

Retornar apenas tags com essas referências

reference-type

Retornar apenas tags com referências desses tipos

page

Retornar apenas o conjunto de resultados de uma página específica (por exemplo 5)

page-size

Modifique o número de itens exibidos por página (por exempo 1 a 50)

show-references

Mostrar dados de referência associados, como ISBNs

exemplo

search?q=tecnologia&tag=science/innovation&web-title=Innovation
